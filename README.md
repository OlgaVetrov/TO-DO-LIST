# ✅ To-Do List

## 📌 Project Description

To-Do List is a simple task management application created with React.

The app helps users organize daily tasks by adding goals, marking completed tasks, and managing their list.

The main focus of this project was creating a smarter delete system. Only completed (crossed) tasks can be deleted, while unfinished tasks stay visible and can be completed later.

I created this project to practice React components, state management, and working with user interactions.

## ⚙️ Installation

To run this project locally, follow these steps:

```bash
git clone https://github.com/OlgaVetrov/TO-DO-LIST.git

cd TO-DO-LIST

npm install

npm start
```

After running the command, the application will open in your browser.

## 🚀 Usage

✅ **To-Do List helps you organize your daily tasks and keep track of your goals.**

Users can add new tasks, mark completed tasks, and keep unfinished tasks visible for later.

### ✨ What you can do:

- ➕ Add new tasks
- ✔️ Mark tasks as completed (crossed)
- 📝 Keep unfinished tasks for the future
- 🗑️ Delete only completed (crossed) tasks

## ⭐ Main Feature

The main idea of this project is the delete logic.

Instead of deleting all tasks immediately, the application allows users to remove only completed (crossed) tasks.

This helps users keep unfinished goals visible and continue working on them later.
